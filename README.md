# 🎨 配色スタジオ

イメージテイストから配色を提案するカラーパレットツール。ビルド不要の静的サイト（HTML / CSS / Vanilla JS）です。

**公開ページ: https://1935hiro.github.io/color-scheme/**

## 機能

- **配色ジェネレーター** — 基準色（HEX / カラーピッカー / HSL スライダー）と 40 種類のイメージテイストを選ぶと、そのテイストの世界観に合った配色レシピ 6 パターン（各 6 色）を生成
  - スウォッチをクリックするとその色を基準にドリルダウン（「前に戻る」対応）
  - カラーコードのコピー、パレットのグラデーション CSS コピー
- **アクセシビリティ** — WCAG 2.1 コントラストチェッカー、基準色×白黒テキストの判定、色覚多様性（P型 / D型 / T型）シミュレーション
- **メタリックグラデーション集** — 金属質感の CSS グラデーション 40 種類

## 使い方

`index.html` をブラウザで開くだけです。

```
open index.html
```

## 配色レシピの仕組み

テイスト定義とレシピは `tastes.js` にあります。各テイストは:

- `s` / `l` — トーンレンジ。レシピの相対位置（0–1）をこの範囲にマップ
- `recipes` — 6 つの配色レシピ（各 6 色のスペック）

色スペックの記法:

| キー | 意味 |
|---|---|
| `b: 1` | 基準色そのまま（各レシピに必ず 1 つ） |
| `dh` | 基準色からの相対色相（度） |
| `hue` | 絶対色相アンカー。`blend`（既定 0.25）の割合で基準色相へ引き寄せ |
| `n: 1` | ニュートラル色（低彩度固定、`l` は絶対レンジ 12–94% へマップ） |
| `s` / `l` | トーンレンジ内の相対位置（0–1、グラデーション用に範囲外も可） |
| `sAbs` | 彩度の絶対値指定（トーンレンジを無視） |

基準色の彩度・明度スライダーの操作は、全レシピの色に緩やかに反映されます。
